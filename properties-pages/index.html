    Properties Pages

    Go to Dashboard → Pages → Add New and provide a page title that suit your purpose.

    Select the appropriate template from page attributes. As examples are displayed in 5 screen shots below.

    You can filter and sort the properties the way you want from Properties Settings meta boxes displayed in the screen shot below.

    If you want to change the default banner then you can do that from Banner Settings meta box displayed in the screen shot below.

    You can Publish the page once it is ready.

    After publishing you need to add the newly created page in menu from Dashboard → Appearance → Menus
